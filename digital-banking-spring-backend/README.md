# EBank - Backend

## Description
Application backend pour système bancaire développée en Java (JEE). Gère les clients, comptes bancaires et opérations financières.

## Technologies
- **Backend**: Java JEE
- **Base de données**: MySQL
- **API**: REST
- **Documentation**: Swagger
